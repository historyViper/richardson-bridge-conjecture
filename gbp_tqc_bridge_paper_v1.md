## 2. The Standard TQC Framework

### 2.1 From Braid Group to Quantum Gates

The braid group Bₙ on n strands is generated by σ₁, …, σₙ₋₁ satisfying:

```
σᵢ σⱼ = σⱼ σᵢ           for |i−j| > 1
σᵢ σᵢ₊₁ σᵢ = σᵢ₊₁ σᵢ σᵢ₊₁   (braid relation)
```

A representation ρ: Bₙ → GL(V) assigns a matrix to each braid generator. If these matrices are unitary and the representation is irreducible, braiding acts as quantum gate operations on the Hilbert space V.

The parameter δ is the loop weight — the value assigned to a contractible closed loop. For SU(2)_k anyons:

The Kauffman bracket assigns a Laurent polynomial in A to any knot or link diagram via the skein relation:

```

⟨○⟩ = −(A² + A⁻²) = δ
```

The Jones polynomial V(t) is obtained from the Kauffman bracket after normalisation by the writhe: V(t) = (−A)^(−3w) ⟨L⟩ where t = A⁻⁴.
